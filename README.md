# Joe Mama Website

Modern React website for Joe Mama, a healthy food business in Laoag City focused on fresh, high-protein, affordable meals.

## Run Locally

Install dependencies:

```powershell
npm install
```

Start the development server:

```powershell
npm run dev
```

Open the local URL printed by Vite, usually:

```text
http://localhost:5173
```

## Project Structure

- `src/App.jsx` - main React website component
- `src/main.jsx` - React entry point
- `src/styles.css` - Tailwind CSS entry file and base styles
- `index.html` - Vite HTML shell with SEO metadata
- `tailwind.config.js` - Tailwind scan configuration
- `postcss.config.js` - PostCSS setup for Tailwind

## Useful Commands

```powershell
npm run dev
npm run build
npm run preview
```
